# Weather App

The Weather App is a Flutter application that allows users to get weather information for different cities.

## Features

- View current weather information for a specific city
- Toggle between light and dark themes
- Search for weather information by entering a city name

## Getting Started

To get started with the Weather App, follow these steps:

1. Clone the repository:

   ```bash
   git clone <repository_url>
   ```

2. Install the dependencies:

   ```bash
   flutter pub get
   ```

3. Run the app:

   ```bash
   flutter run
   ```

## Dependencies

The Weather App relies on the following dependencies:

- [http](https://pub.dev/packages/http): Used to make HTTP requests to fetch weather data
- [cupertino_icons](https://pub.dev/packages/cupertino_icons): Provides the Cupertino icons used in the app
- [intl](https://pub.dev/packages/intl): Used for internationalization and formatting dates

For more information on how to configure and use these dependencies, refer to their respective documentation.
